# Student Marks Analyzer (Python)

A simple console-based Python application to manage student marks and analyze performance.
This project was built to practice core Python concepts like functions, loops, lists, dictionaries, and basic statistics.

## Features
- Add student details with marks for multiple subjects
- View all students with their marks and average score
- Show class statistics:
  - Class average
  - Topper (highest average)
  - Lowest scorer (lowest average)
- Menu-driven command line interface

## Concepts Used
- Functions
- Loops and conditionals
- Lists and dictionaries (nested data structures)
- Aggregation logic (sum, average, max, min)
- f-strings for formatted output

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   ```bash
   python student_marks_analyzer.py
