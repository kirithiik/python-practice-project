# Simple Calculator (Python)

A simple command-line calculator built using Python.
The program performs basic arithmetic operations using a menu-driven interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with division-by-zero handling)
- Menu-driven loop until exit

## Concepts Used
- Functions
- Conditional statements
- While loops
- User input handling
- Basic error checking

## How It Works
- The user selects an operation from the menu
- The program asks for two numbers
- The selected operation is performed
- The result is displayed
- The menu repeats until the user exits

## How to Run
```bash
python calculator.py
