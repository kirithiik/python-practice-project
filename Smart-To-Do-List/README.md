# Smart To-Do List (Python)

A simple console-based To-Do application built using Python.
This project was created to practice core programming concepts such as functions, loops, lists, dictionaries, and state management.

## Features
- Add tasks with title and priority
- View all tasks with status (completed / pending)
- Mark tasks as completed
- Delete tasks
- Menu-driven command line interface

## Concepts Used
- Functions
- Loops and conditionals
- Lists and dictionaries
- Boolean state handling
- Conditional expressions
- f-strings for formatted output

## How to Run
1. Ensure Python is installed on your system.
2. Run the program using:
   ```bash
   python smart_todo.py
