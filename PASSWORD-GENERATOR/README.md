# Password Generator (Python)

A simple command-line password generator built using Python.
The program creates a random password based on a user-defined length using letters, numbers, and symbols.

## Features
- User-defined password length
- Includes uppercase and lowercase letters
- Includes digits and special characters
- Generates a random, secure password

## Concepts Used
- Functions
- Loops
- Random selection
- String manipulation
- Input validation
- Python standard libraries (`random`, `string`)

## How It Works
- The user enters the desired password length
- A pool of characters is created using letters, digits, and symbols
- Random characters are selected to form the password
- The generated password is displayed to the user

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   ```bash
   python password_generator.py
