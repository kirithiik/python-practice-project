# Quiz Application (Python)

A simple command-line quiz application built using Python.
The program presents multiple-choice questions, accepts user input, evaluates answers, and displays the final score.

## Features
- Multiple-choice questions
- Displays options for each question
- Accepts user input for answers
- Instant feedback (correct / wrong)
- Final score summary

## Concepts Used
- Lists and dictionaries
- Loops and conditional statements
- Enumerate for option numbering
- User input handling
- Program flow control

## How It Works
- Questions are stored as a list of dictionaries
- Each question is displayed with numbered options
- The user selects an option by number
- The program checks the answer and updates the score
- Final score is shown after all questions are answered

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   ```bash
   python quiz_app.py
