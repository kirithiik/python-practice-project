# CLI Quiz Engine v2 (Python)

A medium-level command-line quiz application built using Python.
The program allows users to add questions dynamically and take a quiz with score tracking.

## Features
- Menu-driven interface
- Add custom quiz questions
- Multiple-choice questions
- Dynamic question storage
- Score tracking
- Final result summary

## Concepts Used
- Lists and dictionaries
- Functions and modular design
- Loops and conditional logic
- User input validation
- Program flow control

## How It Works
- The user can add quiz questions with four options
- Each question stores the correct answer index
- When the quiz starts, all questions are asked one by one
- The program checks answers and updates the score
- A final score is displayed at the end
