# Number Guessing Game (Python)

A simple command-line number guessing game built using Python.
The program generates a random number within a user-defined range, and the player tries to guess it with hints provided after each attempt.

## Features
- User-defined range for the random number
- Random number generation
- Feedback on each guess (Too High / Too Low / Out of Range)
- Continues until the correct number is guessed

## Concepts Used
- Functions
- Loops
- Conditional statements
- Random number generation (`random` module)
- User input handling

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   ```bash
   python number_guessing_game.py
