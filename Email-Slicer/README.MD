# Email Slicer (Python)

A simple command-line Email Slicer built using Python.
The program extracts the username and domain from a valid email address.

## Features
- Extracts username from email
- Extracts domain from email
- Handles invalid email input

## Concepts Used
- String operations
- Input validation
- Split method
- Program flow control

## How It Works
- The user enters an email address
- The program checks if the email contains '@'
- The email is split into username and domain
- The extracted values are displayed

## How to Run
```bash
python email_slicer.py
